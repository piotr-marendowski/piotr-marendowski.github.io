IvoryOS 1.0 is here!

After a long time (since july) I'm presenting the IvoryOS 1.0!

It's an Arch Linux based GNU/Linux distribution, which runs on x86_64 architecture. It has TUI install wizard, and is more or less versatile!

First it was an install script, then I created a simple Arch install medium with a script that will be executed and do a lot of stuff for me (like installing the base system), and then added an option for it to download my dotfiles into it. Looking back at it, I think that it was well thought (I guess). I didn't plan on making it modular, but made it by chance. Because of it you can install basic Arch without anything and then provide your dotfiles for it - modular design. Thanks to it I can update my dotfiles without the need of updating ISO every single time.

There is still stuff to add to it (and fix :/), but it's more or less ready for 1.0 release.
